# 蔚来一年微博用户运营分析

为小鹏 GX/X9 高端车型用户运营策略提供竞品参考。

**数据范围**：2025-05-09 ~ 2026-05-07（一年） ｜ **数据量**：1,143 条 ｜ **数据源**：[@蔚来](https://weibo.com/u/5675889356) 官方微博

## 文件

| 文件 | 作用 |
|---|---|
| **蔚来_用户运营分析报告.md** | 主报告（执行摘要 + 月度发文 + 运营动作分类 + Top 案例 + 给小鹏的启示） |
| 蔚来_用户运营分析报告.html | 主报告 HTML 版（由 [make_html_report.py](../make_html_report.py) 生成） |
| 蔚来_分类明细.csv | 1143 条微博的标签 + 正文摘要 + 互动数据 |
| analyze.py | 蔚来专属营销动作分类规则 + 报告生成器 |

## 重跑步骤

```bash
# 从仓库根目录运行
python analysis_nio/analyze.py
```

依赖项目根 `weibo_data/蔚来/5675889356.json`（蔚来一年原始数据）。

## 关键发现速读（详见报告）

- 「转评赞抽奖」工厂化：互动激励占比 58.7%
- NIO Life 自有品牌做奖品 + 心智双载体
- 创始人李斌深度内容化：#ET9会客厅# 等节目
- 「跑线」活动 IP 化：丝绸之路换电行、川藏跑线等
- 运营节奏极稳：日均 3.1 条
